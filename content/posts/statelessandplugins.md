## Stateless OS: From an OpenStack Perspective

En esta sesión hablaremos sobre un S.O. stateless, como difiere a otros Sistemas Operativos y qué lo hace especial cuando se configura OpenStack.Habrá un demo para mosotrar la facilidad y rapidez de reiniciar una configuración de OpenStack volviendolo operacional nuevamente.

Slides Link: [Stateless OS: From an OpenStack Perspective](/presentations/Stateless_OS_From_an_Openstack_perspective.pdf)

## Creación de plugins para Tempest

Gracias a un nuevo feature propuesto en la comunidad, podemos crear plugins independientes del código de Tempest para correr test cases personalizados. En esta plática se presentará la estructura básica de un plugin, su instalación y como ejecutarlo para correr nuestras pruebas.
